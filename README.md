## Challenge Back-end

Creación de dos tablas:
- **clientes**
- **libros**

Creación de 2 endpoints de API:
- **api/clientes/**
- **api/clientes/1**
- **/api/libros/**
- **/api/libros/1**

Ejecutar las siguiente Depencias:
- **npm i -g npm@latest**
- **composer install**

Ejecutar Docker
- **docker compose up**

Ejecutar Migraciones y Seed
- **docker exec -ti myapp php artisan migrate**
- **docker exec -ti myapp php artisan db:seed**
